## 前端面试KroInterview文档

> 打造最受欢迎的前端面试文档.

## 是什么

`KroInterview` 是让前端爱好者挑战更高薪资为目的，同时打造最受欢迎的前端面试文档。

这将非常实用，不会因换工作急着在百度上找五花八门的面试题。

查看[快速开始](front.md)了解详情。

<p style="display:none">
## 特性
- 无需构建，写完文档直接发布
- 容易使用并且轻量 (~19kB gzipped)
- 智能的全文搜索
- 提供多套主题
- 丰富的 API
- 支持 Emoji
- 兼容 IE10+
- 支持 SSR ([example](https://github.com/docsifyjs/docsify-ssr-demo))
</p>
